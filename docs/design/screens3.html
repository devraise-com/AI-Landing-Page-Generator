Mobile-first
Base styles target mobile. Desktop layout added via @media (min-width: 768px). Features stacked on mobile → 3-column row on desktop.
Preview area
Scrollable container with the landing page content. On desktop: centered at max-width 480px to simulate mobile-first view. App header and footer stay fixed.

Edit prompt
